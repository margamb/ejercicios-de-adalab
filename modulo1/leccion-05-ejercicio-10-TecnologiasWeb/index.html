Tecnologias web

Las tecnologías Web sirven para acceder a los recursos de conocimiento disponibles en Internet o en las intranets utilizando un navegador. ... Gracias a la forma en que está organizada la World Wide Web (WWW), los usuarios pueden saltar de un recurso a otro con facilidad.

  • HTML HTML es el lenguaje con el que se define el contenido de las páginas web.
  • CSS CSS son las siglas con las que se conoce a las hojas de estilo en cascada
  • JavaScript Javascript es un lenguaje de programación que permite a los desarrolladores crear acciones en sus páginas web .
